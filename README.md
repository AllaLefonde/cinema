# cinema

Статический сайт-галерея "Кино от Димы Конрадта".

Открыть [index.html](27/index.html) в браузере — там список фильмов, каждый ведёт на страницу
своей папки с двумя фотографиями. Название фильма берётся из файла `1_<название>.jpg` в
папке; если у одного фильма несколько папок, между ними появляется навигация «Предыдущий /
Следующий».
